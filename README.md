# MinecraftRecipes⛏️

[**Minecraft Recipes**](https://minecraft-recipes.vercel.app) includes all the craftable recipes of the popular sandbox game _Minecraft_ in the latest version 1.19.1

It was made with **Next.js** and **TailwindCSS**.
